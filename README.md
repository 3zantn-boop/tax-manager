# tax-manager

# 収入・税金かんたん管理
 
SNS収入（PR案件・モニターなど）を記録して、確定申告の目安を把握するためのツールです。
 
## 使い方
 
HTMLファイルをブラウザで開くだけで使えます。インストール不要。
 
## 主な機能
 
- 一時所得・雑所得の収入を記録
- 経費の記録
- 課税所得と予想税額の自動計算
- 20万円ライン・48万円ラインのアラート表示
- データはブラウザに自動保存（削除しない限り消えません）
## 注意事項
 
- このツールはあくまで目安です。正確な税額は税理士や国税庁サイトでご確認ください。
- データはお使いのブラウザにのみ保存されます。ブラウザのデータを削除すると消えます。
- iPhoneの場合はSafariで開いてください。
